特点：

  • 封装、复用
  • 可以接收参数，也可以返回数据
  • 减少网络交互，效率提升

存储函数

  • characteristic
    • deterministic 系统的输入参数总是产生相同的结果
    • no sql 不包含SQL语句
    • reads sql data 包含读取数据的语句，但不包含写入数据的语句

触发器

触发器能在insert/update/delete之前之后，触发并执行触发器中定义的SQL语句集合。触发器使用OLD和NEW来引用触发器中发生变化的记录内容。MySQL只支持行级触发器，不支持语句级触发器。
